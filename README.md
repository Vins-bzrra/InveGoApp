
<h1>InveGoAPP</h1> 

![Flutter](https://img.shields.io/badge/Flutter-%2302569B.svg?style=for-the-badge&logo=Flutter&logoColor=white)

O InveGo APP foi desenvolvido como uma proposta de interface para utilização do [InveGoAPI](https://github.com/Vins-bzrra/InveGo).

## 🔨 Funcionalidades do projeto
O App realiza todas as funcionalidades disponíveis na API.

## 📁 Acesso ao projeto

Você pode [acessar o código fonte do projeto inicial](https://github.com/Vins-bzrra/InveGoApp) ou [baixá-lo](https://github.com/vins-bzrra/InveGoApp/archive/refs/heads/main.zip).

## 🛠️ Build

 Para utilizar o aplicativo no seu smartphone, você deve fazer a compilação do projeto para gerar o instalador conforme a plataforma, caso você utilize ´Android´, deve executar o seguinte comando:

 ```shell
 flutter build apk
 ```

 Com esse comando será criado o arquivo `.apk` do projeto para realizar a instalação no seu dispositivo. Agora se o seu dispositivo for ´IOS´, deve ser executado o seguinte comando:

```shell
 flutter build ipa
 ```
